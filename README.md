# MulTasker-AppWeb
Una App Web diseñada para cubrir tres areas de trabajo; Economia(registrar ingresos y gastos), Calendario(un calendario para registrar eventos, cumpleaños, etc) y CutClips(recortador de audios mp3).

Cambios:
- Boton para eliminar evento en Calendario
- Cambiar campo de recorte en CutClips
